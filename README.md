# AI-Powered Semantic Search System for Product Reviews 🔍🤖

This project demonstrates how **Text Embeddings and Semantic Search** can be used to retrieve meaningful product reviews based on the user's query instead of relying only on keyword matching.

The system uses **Sentence Transformers and FAISS vector search** to find the most relevant reviews from a large dataset of Flipkart product reviews.

---

## 🚀 Project Overview

Traditional search systems rely on keyword matching which often fails when users use different words to express the same intent.

This project implements an **AI-powered semantic search system** that understands the **meaning of text** using embeddings and retrieves relevant product reviews.

## 🔎 How Semantic Search Works

1️⃣ Convert reviews into **vector embeddings** using Sentence Transformers  
2️⃣ Store embeddings in a **FAISS vector index**  
3️⃣ Convert user query into embedding  
4️⃣ Retrieve the most similar reviews using vector similarity
